Repetitiveness of a song

This webside can show you the repetitiveness of a song using a Self-similarity matrix or a Dot plot. The idea of this project came from Colin Morris "Tedx Talk" Pop Music is Stuck on Repeat.

How to use

  1. Paste the song text in the textfield.
  2. Choose if each dot should represent a word or a line.
  3. Set the size of the dots.
Also you can export the matix to an image if you like to share it.
